# LabATM
This file is a basic code for a bank account at an ATM. 
It asks user how much money they have. 
Allows user to withdraw a maximum of that ammount. Gives error if over drawn.
If 0 is input the code terminates.

## How to run
1. Open program in IDE
2. Enter starting balance
3. Enter deposit amount
4. Continue to enter another withdrawal until 0 or input 0 to finish.

## Example output
Enter starting balance: 200
Enter amount to withdraw (0 to exit): 50
Withdrew $50.0Balance = $150.0
Enter amount to withdraw (0 to exit): 0
Goodbye!
